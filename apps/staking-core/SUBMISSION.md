## Project Summary
A full staking protocol deployed on the QIE Testnet.
Users lock tokens, earn yield, and claim minted rewards.

Focused on real adoption: low gas, secure, modular, and compatible with any ERC-20 token.

------------------------------------------------------------

## Key Features
- Stake/unstake system with lock period
- Reward token minting (hard-capped)
- Optional oracle-driven APY
- Full Hardhat testing suite
- Clean vanilla-js frontend
